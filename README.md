# Aprendizado de Máquina

Repositório com trabalhos desenvolvidos na disciplina de **Aprendizado de Máquina**.


## Projetos

### 1. Otimização de Pipeline

Notebook voltado à otimização de um pipeline de Machine Learning, com foco em organização do fluxo de pré-processamento, treinamento e avaliação de modelos.

**Principais tópicos abordados:**

- Construção de pipeline de aprendizado de máquina
- Pré-processamento de dados
- Treinamento e comparação de modelos
- Avaliação de métricas de desempenho
- Otimização do fluxo experimental

**Notebook:**

### 2. Modelagem Preditiva de Saúde Mental

Projeto focado na aplicação de modelos preditivos para análise de dados relacionados à saúde mental, buscando identificar padrões e realizar previsões com base nas variáveis disponíveis.

**Principais tópicos abordados:**

- Análise exploratória de dados
- Preparação e limpeza da base
- Modelagem preditiva
- Avaliação de modelos de classificação
- Discussão dos resultados obtidos
